Documentation - Suivi de Réseau¶

Bienvenue dans la documentation du projet Suivi de Réseau, un outil de scan d’adresses IP et de ports TCP, conçu pour analyser des réseaux locaux en profondeur.

Ce projet permet de : - Scanner une plage IP en CIDR ou un fichier CSV - Identifier les hôtes actifs (via ping) - Scanner les ports ouverts (avec Nmap) - Exporter les résultats en CSV
